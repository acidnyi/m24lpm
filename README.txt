The data and models is already provided in the folders "data" and "models"

If you want to install the data by yourself, use the download_data.py file

Also, don't forget to install requirements provided in the requirements.txt

To reproduce the project:

1. Run the tidying.R file, this will create a data.csv

2. Run the eda.R, this file also contains the scenario 1 and 3, and files needed for the shiny app

3. Run the app.R to get a shiny app.

link to the github: https://github.com/acidnyi/m24lpm
